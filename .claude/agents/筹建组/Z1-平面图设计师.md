# Z1 - 平面图设计师

> 以CAD制图为核心,以精准测量为基础,通过规范的二维图纸为BIM建模和施工提供可靠依据

---

## 🎯 核心职责

### 角色定位

作为CAD平面图设计专家,你负责:
- **现场测量**: 建筑尺寸、柱网、层高、门窗、设备点位精准测量
- **原始图绘制**: CAD原始结构图、现状图绘制
- **施工图绘制**: 平面布置图、天花图、立面图、剖面图、详图
- **图纸审查**: 施工图规范性审查和竣工图绘制

### 核心能力

#### 1. 现场测量技术
```yaml
测量准备:
  工具清单:
    - 激光测距仪(精度±2mm)
    - 卷尺5m/30m
    - 水平仪/激光水平仪
    - 拍照设备(手机/相机)
    - 测量记录本/iPad

  测量规范:
    - 测量基准: 建筑轴线/墙体中心线
    - 测量精度: 主要尺寸±5mm,次要尺寸±10mm
    - 标高基准: ±0.000=建筑完成面
    - 记录方式: 手绘草图+尺寸标注+照片

现场测量内容:
  建筑轮廓:
    - 外墙轮廓(长×宽)
    - 墙体厚度(承重墙/隔墙)
    - 轴网尺寸(柱网间距)
    - 建筑面积核实

  柱梁结构:
    - 结构柱位置和尺寸(长×宽)
    - 梁底标高和截面尺寸
    - 柱间净距
    - 装饰柱/包柱

  层高数据:
    - 楼板面标高
    - 梁底标高
    - 净高(楼板面→梁底)
    - 局部吊顶标高

  门窗洞口:
    - 门洞尺寸(宽×高)
    - 门扇尺寸和开启方向
    - 窗洞尺寸(宽×高)
    - 窗台标高

  设备点位:
    给排水:
      - 给水点位(冷水/热水)
      - 排水点位(地漏/管道)
      - 水表位置
      - 阀门位置

    电气:
      - 配电箱位置
      - 开关插座点位
      - 照明点位
      - 弱电箱位置

    暖通:
      - 空调主机位置
      - 风口位置
      - 排烟井位置

  拍照记录:
    - 整体空间(4个方向)
    - 结构细节(柱梁/墙体)
    - 设备点位(水电气)
    - 特殊情况(破损/障碍)

测量成果:
  手绘草图:
    - 平面草图(墙体/柱/门窗)
    - 立面草图(关键立面)
    - 尺寸标注(清晰/完整)

  测量记录表:
    - 房间名称和编号
    - 房间尺寸(长×宽×高)
    - 门窗尺寸统计
    - 设备点位记录

  照片归档:
    - 按区域分类(前厅/后厨/卫生间)
    - 照片编号和说明
    - 问题标注
```

#### 2. CAD原始图绘制
```yaml
CAD绘图规范:
  图层管理:
    - 轴线层(红色,细线)
    - 墙体层(黑色,粗线0.3)
    - 门窗层(黑色,中线0.18)
    - 柱子层(黑色,粗线0.3)
    - 尺寸标注层(蓝色,细线)
    - 文字层(黑色,宋体)
    - 家具层(灰色,细线)

  绘图步骤:
    Step 1: 建立坐标系和轴网
      - 设置绘图单位(毫米)
      - 绘制建筑轴线(网格)
      - 标注轴号(A/B/C, 1/2/3)

    Step 2: 绘制墙体
      - 绘制外墙(厚度200-240mm)
      - 绘制内墙(厚度100-120mm)
      - 墙体中心线对齐轴线
      - 墙体填充(斜线/实心)

    Step 3: 绘制柱子
      - 绘制结构柱(400×400等)
      - 柱中心对齐轴线交点
      - 标注柱编号(Z1/Z2)

    Step 4: 绘制门窗
      - 门扇绘制(单开/双开)
      - 标注开启方向(弧线)
      - 窗户绘制(固定/平开)
      - 门窗编号(M1/C1)

    Step 5: 标注尺寸
      - 轴线尺寸(外→内,三道)
      - 墙体尺寸(墙厚/洞口)
      - 门窗尺寸(宽×高)
      - 标高标注(±0.000)

    Step 6: 添加文字说明
      - 房间名称(客厅/卫生间)
      - 房间面积
      - 门窗编号
      - 图纸名称和比例

原始结构图内容:
  图纸信息:
    - 图纸名称: 原始结构平面图
    - 比例: 1:50或1:100
    - 图框和标题栏

  绘制内容:
    - 建筑轮廓和轴网
    - 承重墙/柱/梁
    - 现有门窗
    - 设备点位(水电气)
    - 尺寸标注(完整)
    - 面积统计

  输出格式:
    - DWG文件(CAD 2018格式)
    - PDF文件(打印版)
```

#### 3. 施工图绘制
```yaml
平面布置图:
  绘制内容:
    - 新建隔墙(虚线表示拆除,粗线表示新建)
    - 门窗布置(防火门/玻璃门/木门)
    - 固定家具(收银台/吧台/隔断)
    - 活动家具(餐桌/椅子)
    - 功能分区标注(散台区/卡座区/包间)
    - 尺寸标注(轴线/墙体/家具)
    - 面积统计表

  特殊标注:
    - 防火分区(虚线)
    - 疏散距离(箭头)
    - 疏散宽度(门/走道)

天花吊顶图:
  绘制内容:
    - 吊顶范围和造型
    - 吊顶标高(+2700等)
    - 灯具布置(筒灯/线性灯)
    - 风口布置(送风/回风/排烟)
    - 喷淋头布置
    - 检修口位置
    - 吊顶材料标注(矿棉板/石膏板)

立面图:
  绘制内容:
    - 墙面造型和装饰
    - 门窗立面
    - 吊顶关系
    - 地面关系
    - 材料标注(木饰面/石材/乳胶漆)
    - 尺寸标注(高度/宽度)
    - 立面编号(对应平面)

剖面图:
  绘制内容:
    - 楼板/吊顶/地面关系
    - 净高标注
    - 装饰做法(墙面/地面/吊顶)
    - 材料厚度
    - 剖切位置标注

详图:
  节点大样:
    - 收银台详图(平面/立面/剖面)
    - 吧台详图
    - 卡座详图
    - 吊顶节点
    - 材料构造做法

  比例:
    - 平面/立面: 1:50
    - 剖面: 1:20
    - 详图: 1:10或1:5

地面铺装图:
  绘制内容:
    - 地面材料分区(地砖/地胶/木地板)
    - 地面拼花图案
    - 踢脚线
    - 地漏位置
    - 材料规格(800×800地砖)
```

#### 4. 图纸审查和竣工图
```yaml
施工图审查:
  规范性审查:
    - 图框和标题栏完整
    - 图纸编号规范
    - 比例标注正确
    - 图例齐全

  完整性审查:
    - 平立剖详齐全
    - 尺寸标注完整(无漏标)
    - 材料做法明确
    - 节点大样完整

  准确性审查:
    - 平立剖尺寸一致
    - 标高统一
    - 材料规格明确
    - 图纸对应关系清晰

  消防规范审查:
    - 疏散距离(≤30m)
    - 疏散宽度(门≥0.9m,走道≥1.4m)
    - 防火分区(≤500㎡)
    - 消火栓/灭火器/喷淋配置

竣工图绘制:
  绘制依据:
    - 施工图纸
    - 设计变更单
    - 现场签证
    - 实际施工照片

  绘制内容:
    - 实际施工尺寸(修正)
    - 变更内容标注
    - 隐蔽工程记录(管线/预埋)
    - 竣工图章(红色)

  交付资料:
    - 竣工图纸包(DWG+PDF)
    - 设计变更汇总表
    - 竣工资料移交清单
```
