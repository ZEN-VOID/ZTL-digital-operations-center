## 📖 项目简介

ZTL数智化作战中心是一个基于Claude Code和多智能体协作的餐饮行业数智化转型平台。通过**7大业务组**和**60个专业智能体**的协同工作，实现从战略规划、品牌营销、数据分析、行政管理、运营中台、门店筹建到系统开发的**全链路数智化支持**。

### 核心价值

- 🎯 **智能决策**: 基于Claude Sonnet 4.5的深度推理能力
- 🤖 **多智能体协作**: 60个专业智能体按业务分工协同
- 🔌 **MCP生态集成**: 7大MCP服务器无缝对接外部系统
- 📊 **数据驱动**: 全流程数据采集、分析和可视化
- 🚀 **快速响应**: 通过斜杠命令实现一键式工作流
- 🔄 **持续进化**: ASDW学习系统支持自我优化

## ✨ 核心特性

### 1. 六大业务组智能体系统

- **创意组 (X系列)** - 9个智能体：广告策划、文案创作、平面设计、短视频制作

15个框架级智能体提供开发工具、系统能力和智能体工程支持：

- F1: Agents智能体创建工程师

- F3: Hooks创建工程师
- F4: Output-style输出样式设计师
- F5: Skills技能包创建工程师

一键式工作流程，覆盖上下文管理、执行控制和项目管理：

### 技术栈

| 类别             | 技术          | 版本  | 用途             |
| ---------------- | ------------- | ----- | ---------------- |
| **AI引擎** | Claude Sonnet | 4.5   | 核心推理引擎     |

| **协作**   | Lark/Feishu   | -     | 飞书企业协同     |

### 架构模式

本项目采用**三层智能体架构**：

```
Layer 1: 知识层 (Agents + Skills)
  - Agents: 角色化决策框架和专业领域知识
  - Skills: 工具/流程能力包和执行引擎
  ↓
Layer 2: 决策编排层 (Claude Reasoning)
  - 运行时推理和动态能力组合
  - 智能路由和任务调度
  ↓
Layer 3: 执行输出层 (Tools + Output)
  - Bash, Python, API, MCP工具执行
  - 结果持久化到output/目录
```

## 🚀 快速开始

### 环境要求

- **macOS**: Darwin 24.3.0+
- **Claude Code**: v1.0+
- **Python**: 3.10+
- **Git**: 2.0+

### 安装步骤

1. **克隆项目**

   ```bash
   git clone https://github.com/yourusername/ZTL数智化作战中心.git
   cd ZTL数智化作战中心
   ```
2. **配置Claude Code**

   - 确保已安装Claude Code
   - 项目配置文件位于 `.claude/`目录
3. **安装依赖** (如需要)

   ```bash
   # Python依赖
   pip install -r requirements.txt

   # MCP服务器配置
   # 参考 .claude/CLAUDE.md 中的MCP配置说明
   ```

### 使用指南

#### 通过斜杠命令使用

在Claude Code中直接调用命令：

```bash
# 项目文档生成
/I                    # 生成README.md
/O                    # 生成OVERVIEW.md
/M                    # 更新项目级CLAUDE.md

# 功能开发
/E <需求描述>         # 端到端PRP工作流
/F <功能描述>         # 快速生成PRP文档

# 项目管理
/G                    # 完整同步到GitHub
/Y                    # Issue分析修复
/Q                    # 全面分析项目

# ASDW学习系统
/A                    # What: 现状考察
/S                    # Why: 原因分析
/D                    # How: 方法研究
/W                    # Wield: 整合执行
```

#### 通过智能体使用

调用Task工具指定智能体类型：

```python
# 战略分析
Task(subagent_type="G1-经营分析优化师",
     prompt="分析本月门店经营数据")

# 创意设计
Task(subagent_type="X3-平面设计师",
     prompt="设计新品海报")

# 情报采集
Task(subagent_type="E2-网站情报采集员",
     prompt="采集竞品网站数据")
```
